#installing:
1) install NodeJS from their website
    -default options should be fine
2) install ruby (mac may already have this)
    -make sure to add ruby to path when installing
3) in Webstorm create new project from git which should theoretically pull down everything...
3) in the server folder from the command line/terminal run:
    - npm install
4) in the client folder from the command line/terminal run:
    - npm install
    - bower install
